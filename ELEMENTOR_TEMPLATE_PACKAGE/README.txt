🚀 QUICK START - 3 PASSOS:

1️⃣ Upload nuclea-menu-vanilla.js → /wp-content/themes/tema/assets/

2️⃣ Import elementor-template.json no Elementor

3️⃣ Configure JavaScript via WPCode (código no guia)


📖 COMO USAR:

┌──────────────────────────────────────┐
│  Passo 1: Abra LEIA_PRIMEIRO.md     │  ← Comece por aqui!
│           (5 minutos de leitura)     │
├──────────────────────────────────────┤
│  Passo 2: Siga os 3 passos           │  (15 minutos)
├──────────────────────────────────────┤
│  Passo 3: Customize (opcional)       │  (10 minutos)
│           Use CUSTOMIZACAO.md        │
└──────────────────────────────────────┘


✅ REQUISITOS:

- WordPress 6.0+
- Elementor 3.0+ (Free ou Pro)
- Plugin WPCode (recomendado) ou acesso ao functions.php
- Acesso FTP/cPanel


⏱️ TEMPO ESTIMADO:

Implementação básica:  15-20 min
Com customização:      25-30 min


🎯 O QUE VOCÊ TERÁ:

✅ Menu mega hierárquico (até 4 níveis)
✅ Design profissional e responsivo
✅ Animações suaves
✅ Seletor de idiomas
✅ Botão CTA customizável
✅ Zero dependências


📝 ORDEM DE LEITURA:

1. LEIA_PRIMEIRO.md          ← Essencial! Comece aqui
2. (Implementar)              
3. CUSTOMIZACAO.md           ← Para personalizar
4. VISUAL_GUIDE.md           ← Consulta rápida (opcional)
5. ELEMENTOR_IMPORT_GUIDE.md ← Se tiver dúvidas (opcional)


🆘 PROBLEMAS?

1. Verifique LEIA_PRIMEIRO.md → "Problemas Comuns"
2. Consulte ELEMENTOR_IMPORT_GUIDE.md → "Troubleshooting"
3. Abra Console do navegador (F12) para ver erros


💡 DICAS:

✓ Faça backup antes de começar
✓ Teste em staging se possível
✓ Limpe cache após mudanças (Ctrl+F5)
✓ Valide JSON ao editar itens do menu
